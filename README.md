\# 🎙 Alexa Voice Assistant in Python



A simple Alexa-like voice assistant built using Python.



\## 🚀 Features

\- 🎵 Play songs on YouTube

\- ⏰ Tell current time

\- 📖 Wikipedia search

\- 😂 Tell jokes



\## 🛠 Technologies Used

\- Python

\- SpeechRecognition

\- pyttsx3

\- pywhatkit

\- wikipedia

\- pyjokes



\## 📦 Installation



Install required libraries:



pip install -r requirements.txt



\## ▶️ Run the Program



py alexa.py
